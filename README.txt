K-R AI Command Center

Upload this folder to GitHub, then import it to Vercel.

Vercel Environment Variables:
GEMINI_API_KEY = your new Gemini key
GEMINI_MODEL = gemini-1.5-flash

Optional Supabase:
SUPABASE_URL
SUPABASE_SERVICE_ROLE_KEY

Optional phone control:
JOIN_API_KEY
JOIN_DEVICE_ID

Supabase:
Run supabase_schema.sql in Supabase SQL Editor.

Phone:
Read tasker/TASKER_SETUP.txt

Laptop:
Run laptop-agent/kr_agent.py later.

Never put API keys in GitHub or index.html.
